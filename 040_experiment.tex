\section{Эксперимент}
\label{sec:experiment}

\subsection{Цель и методология}

Целью эксперимента является количественная оценка точности автоматических морфометрических измерений
системы путём сравнения с ручными измерениями, выполненными специалистом лаборатории Гербарий ГБС РАН.

Тестовая выборка включает 9~изображений листьев \textit{Sphagnum} из гербарных коллекций.
Для каждого изображения специалист выполнил ручное измерение двух морфометрических характеристик:
\begin{itemize}
    \item длины листа по главной оси;
    \item поперечных профилей ширины в семи равноотстоящих сечениях
          (на долях 0.125, 0.25, 0.375, 0.5, 0.625, 0.75, 0.875 от полной длины).
\end{itemize}

Те же изображения были обработаны разработанной системой с минимальным числом подсказок~---
как правило, один-два положительных клика по телу листа.
Автоматические маски сохранены в директории \texttt{test/results/};
ручные эталонные маски~--- в \texttt{test/mapped/}.

\subsection{Метрики качества}

Для оценки погрешности измерений использовались следующие метрики:
\begin{align*}
    \text{Bias} &= \frac{1}{N}\sum_{i=1}^{N}(x_i^\text{авт} - x_i^\text{руч}),
    &\text{MAE} &= \frac{1}{N}\sum_{i=1}^{N}|x_i^\text{авт} - x_i^\text{руч}|, \\
    \text{RMSE} &= \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i^\text{авт} - x_i^\text{руч})^2},
    &\text{MAPE} &= \frac{1}{N}\sum_{i=1}^{N}\frac{|x_i^\text{авт} - x_i^\text{руч}|}{x_i^\text{руч}},
\end{align*}
где $x_i^\text{авт}$~--- автоматическое измерение, $x_i^\text{руч}$~--- ручное измерение.
Дополнительно вычислялся коэффициент корреляции Пирсона~$R$ между автоматическими и ручными значениями.

Положительный Bias означает систематическое завышение автоматических значений относительно ручных.

несколько удлиняя итоговое значение.

пикселя маски, а не для систематической ошибки метода.

PCA-алгоритма: главная ось ориентируется по облаку пикселей маски и при наличии широкого
основания или артефактов краевых пикселей несколько удлиняется.
Данная погрешность стабильна и может быть скорректирована калибровочным коэффициентом.

\subsection{Апробация}

Прототип системы был продемонстрирован консультанту~--- м.н.с.\ лаборатории Гербарий ГБС РАН
Шкурко~А.~В. По итогам демонстрации качество автоматической сегментации признано
удовлетворительным для дальнейшего развития системы.
Видеозапись демонстрации прилагается к работе.

В ходе апробации были выявлены следующие направления доработки:
\begin{itemize}
    \item поддержка масштабного коэффициента (пересчёт пикселей в мм);
    \item возможность экспорта визуализаций в дополнение к CSV;
    \item расстановка точек вдоль периметра с заданным шагом (требование из технического задания).
\end{itemize}
