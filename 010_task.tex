\section{Постановка задачи}
\label{sec:task}

\subsection{Цель и задачи работы}

Работа выполняется в интересах лаборатории Гербарий ГБС РАН, сотрудники которой проводят
морфометрический анализ листьев мохообразных (в первую очередь мхов рода \textit{Sphagnum}).

Рабочий процесс биолога устроен следующим образом. На каждом снимке присутствует несколько
листьев разного качества, и для дальнейших измерений отбираются «хорошие» листья~---
целые, с неповреждёнными краями, типичной для вида формы, расположенные изолированно и не
перекрывающиеся с другими объектами. После отбора каждый такой лист сегментируется
вручную и по полученной маске извлекаются геометрические параметры~--- длина и ширина
в нескольких поперечных сечениях. При сериях из десятков и сотен снимков обе операции~---
отбор и измерение~--- становятся узким местом: занимают значительное время и вносят
субъективную погрешность, особенно когда работу выполняют несколько человек.

\textbf{Цель работы}~--- разработать настольное приложение, полностью автоматизирующее
цикл «загрузка снимков $\to$ сегментация всех листьев $\to$ отбор пригодных $\to$
параметрические измерения $\to$ экспорт» для морфометрии листьев \textit{Sphagnum}.
В случаях, когда полная автоматизация неприменима (например, для вида листа, по которому
модель отбора ещё не обучена), приложение должно поддерживать интерактивный режим работы
с возможностью ручной коррекции масок. Приложение должно работать на стандартных рабочих
станциях лаборатории без выделенного GPU.

Для достижения цели поставлены следующие \textbf{задачи}:
\begin{enumerate}
    \item Проанализировать существующие программные решения для сегментации и морфометрии
          биологических объектов и обосновать выбор моделей сегментации и отбора.
    \item Сформулировать требования к системе совместно с заказчиком~--- сотрудниками
          лаборатории Гербарий ГБС РАН.
    \item Спроектировать архитектуру приложения и реализовать его, включая интерактивный
          и автоматический режимы работы.
    \item Реализовать алгоритм параметрических измерений (длина, поперечные профили ширины).
    \item Собрать и разметить обучающую выборку для классификатора отбора листьев.
    \item Обучить модель отбора корректных листьев для нескольких видов \textit{Sphagnum}
          и оценить её качество.
    \item Сравнить результаты автоматических измерений с ручной разметкой биологов
          и провести апробацию системы.
\end{enumerate}

\textbf{Практическая значимость} работы состоит в сокращении времени, затрачиваемого
на получение морфометрических данных, и в повышении воспроизводимости измерений за счёт
стандартизованного алгоритма обработки.
